# 项目简介

基于 SpringBoot 开发的测试 demo，具备以下功能：

- SpringBoot 框架简单使用，Controller 各类请求使用
- 集成 MyBatis-Plus、dynamic-datasource,操作数据库及多数据源操作的使用
- 支持 MySQL、PostgreSQL、Oracle 数据库的使用
